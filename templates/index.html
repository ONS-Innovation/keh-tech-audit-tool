What is the Tech Radar Data Collection Tool?

This tool is used to collect data about various projects, which is then utilized by the Tech Radar Tool. This helps in assessing and visualizing the technology landscape within the organization, aiding in strategic decision-making and technology adoption.

Selected users can sign in to the tool using their Cognito credentials. Once signed in, they can create a project and view existing project data.
